Allows the user to specify the default options for the progressCircular directive.

Usage

myAppModule.config(function($mdProgressCircularProvider) { // Example of changing the default progress options. $mdProgressCircularProvider.configure({ progressSize: 100, strokeWidth: 20, duration: 800 }); });

Properties

Property Type Description
* progressSize number number

Diameter of the progress circle in pixels.
* strokeWidth number number

Width of the circle's stroke as a percentage of the circle's size.
* duration number number

Length of the circle animation in milliseconds.
* easeFn function function

Default easing animation function.
* easingPresets object object

Collection of pre-defined easing functions.
* durationIndeterminate number number

Duration of the indeterminate animation.
* startIndeterminate number number

Indeterminate animation start point.
* endIndeterminate number number

Indeterminate animation end point.
* easeFnIndeterminate function function

Easing function to be used when animating between the indeterminate values.
* configure function(object): object function(object): object

Used to modify the default options.
